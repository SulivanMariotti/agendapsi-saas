# Lembretes e notificações — AgendaPsi

Atualizado: **2026-03-02**

## Contexto
- O AgendaPsi é um SaaS separado do Lembrete Psi.
- Notificações (FCM/WhatsApp/SMS) serão implementadas apenas dentro do AgendaPsi, quando aplicável.

## MVP
- Sem FCM no MVP (a menos que seja necessário para confirmação).
- Botão de WhatsApp no detalhe do agendamento/hold (mensagem reforçando presença/constância).

## Pós-MVP
- Templates de WhatsApp no Admin + preenchimento automático.
- Notificações programadas (FCM/email) com foco em constância, evitando padrões que incentivem cancelamento.
